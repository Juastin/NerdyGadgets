NerdyGadgets project of group 2.

To run the webshop, the simple database must first be imported.
To do this follow the following steps:

1.  Run xampp (downloadable here: https://www.apachefriends.org/index.html)
2.  Run apache and Mysql in xampp
3.  Go to http://localhost/phpmyadmin/ and make a new database, name this database: nerdgygadgets_simple
4.  import the nerdygadgets_simple importfile into the newly made database
    by selecting Import at the top when working in the new database, after this choose the import file (found in NerdyGadgets/database/ )
5.  When successfully imported go to http://localhost/nerdygadgets/ and the website will show itself correctly.
